# Aircall Test

## Used environment

-   react 17.0.2
-   react-scripts 5.0.1
-   node v16.17.1
-   yarn 1.22.19
-   typescript 4.8.4

## How to start the app

-   **Install dependencies**: npm install
-   **Run the app**: npm run start
-   **Run tests**: npm run test
-   **Run linter**: npm run lint
-   **Run formatter**: npm run format
-   **Run type checker**: npm run typecheck
